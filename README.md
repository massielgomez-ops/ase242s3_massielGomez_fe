¿Qué es Angular?

Angular es una herramienta que sirve para crear páginas y aplicaciones web modernas. Fue creada por Google. Con Angular se pueden hacer páginas que cambian su contenido sin recargar todo el sitio.

Trabaja con partes llamadas componentes, que ayudan a organizar mejor el código. Gracias a eso, las aplicaciones son más ordenadas, rápidas y fáciles de mantener.

¿Qué herramientas necesitan instalar?

Para usar Angular se necesita instalar primero Node.js, que permite que funcione en la computadora.

Luego se instala Angular CLI, que ayuda a crear y manejar proyectos.

También se necesita un editor de código como Visual Studio Code para escribir el programa, y un navegador como Google Chrome para ver la aplicación.

Con esas herramientas ya se puede empezar a trabajar con Angular.
